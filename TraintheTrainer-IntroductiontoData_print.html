Introduction to Data

When someone mentions the word data, what does it mean? Data are an organized collection of facts about the real world. You can have different types and forms of data. For example, you can have numbers, letters, and words (also known as 'strings' in computer science). These different data types allow us to store complex amounts of information and facts about the world around us.

Now that we understand what data is and the different types of data, how can we store it?

Data can be stored in a variety of ways, and you may already be familiar with some of them. Some ways include in an excel spreadsheet, in a database, in text files, in the .JSON format, log files, and more. Below is an example of how student data can be stored in an excel spreadsheet. We have rows and columns that help define what we are looking at. The rows contain each data point that we're trying to capture. In this case, each row contains data about one student. The columns represent different facets of information that we would like to keep track of regarding the students.

 

As previously mentioned, we can store different data types. In this case, we see both numbers and words (strings). We see words (strings) listed in the Name column, as well as the Letter Grade column. The rest of the columns contain numbers that define the class and grades that the students got.

Now that we have organized and stored these facts into this tabular format, why did we do it? By organizing this data in this manner, we can gain further information about the class and our students. We can utilize the average feature to help define how well each student did overall in the class. We can utilize graphs to determine the spread of grades on particular assignments. Finally, we can keep track of how well the class as a whole did and compare it to future classes to understand if the material taught or graded needs to be adjusted.
